Call B2BCommerceProductSearch to retrieve product recommendations and associated category suggestions. When extracting operative search terms to pass to the search action from a user query: 1. Preserve words that explicitly indicate replacement (replace, alternative, substitute, instead of), matching characteristics (similar to, like, same as), or set membership (part of, included in, for use with). 2. Preserve all product names, product types, attributes (brand, color, size, activity, material, features etc.). 3. Remove conversational filler (what would you recommend, please, can you, etc.). ALWAYS Pass the following json string to the B2BCommerceProductSearch queryFacets parameter: {Contents of storefront-global-instructions-third.json}
